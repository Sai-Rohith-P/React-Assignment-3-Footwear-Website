# Footwear E-Commerce Website

A responsive React.js-based e-commerce application designed for browsing and purchasing footwear products. The application provides an intuitive shopping experience with product listings, category filtering, cart management, and a modern user interface optimized for all devices.

## Features

* Responsive User Interface
* Product Listing
* Product Details View
* Category-Based Filtering
* Shopping Cart Functionality
* Dynamic Cart Updates
* Quantity Management
* Price Calculation
* Checkout / Payment Interface
* Mobile, Tablet, and Desktop Support

## Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Bootstrap

## Key Functionalities

### Product Management

* Browse Footwear Products
* View Product Details
* Filter Products by Category

### Shopping Cart

* Add Products to Cart
* Remove Products from Cart
* Update Product Quantity
* Dynamic Total Price Calculation

### Checkout

* Review Selected Products
* Payment Page Interface

### Responsive Design

* Mobile Friendly
* Tablet Friendly
* Desktop Friendly

## Installation

### Clone Repository

```bash
git clone https://github.com/Sai-Rohith-P/React-Assgnment-3-Footwear-Website.git
```

### Setup

```bash
npm install
npm start
```
